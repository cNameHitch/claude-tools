# Claude Tools

A collection of custom skills and agents for Claude Code.

## Structure

```
claude-tools/
├── skills/     # Custom skills for Claude Code
├── agents/     # Custom agent configurations
└── templates/  # Starter templates
```

## Skills

Skills are reusable prompts that can be invoked with `/skill-name` in Claude Code. See [skills/README.md](skills/README.md) for details.

## Agents

Agents are specialized configurations for handling specific types of tasks. See [agents/README.md](agents/README.md) for details.

## Usage

Copy the desired skill or agent configuration to your Claude Code setup directory.
